# monte-carlo-sdl2
## Requirements
### Ubuntu
```
sudo apt install gcc build-essential libsodium-dev libsdl2-dev
```
### Arch
```
sudo pacman -S sdl2 libsodium
```
## Installation
```
git clone https://github.com/tirex03/monte-carlo-sdl2.git
cd monte-carlo-sdl2
./configure
make
sudo make install
```
## Usage
```
montecarlo
```
